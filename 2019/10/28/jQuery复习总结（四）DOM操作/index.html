jQuery DOM 操作


jQuery 中非常重要的部分，就是操作 DOM（Document Object Model） 的能力

test()、html() and val()

  • text() - 设置或返回所选元素的文本内容
  • html() - 设置或返回所选元素的内容（包括 HTML 标记）
  • val() - 设置或返回表单字段的值

获取内容：

<html>
    <head>
        <title>JQuery DOM Manipulation</title>
    </head>
    <body>
        <div class="hello"><p>这是返回文本内容</p></div>
        <div class="world"><p>这是返回包括HTML文本</p></div>
        <p><input type="text" class="test" value="Haris is a handsome boy"></input></p>
        <button>Click me</button>
    </body>
    <script src="../../js/jquery-3.2.1.min.js"></script>

    <script type="text/javascript">
    $("document").ready(function() {
       $('button').on('click', function(){
         alert('Text: '+$('.hello').text());
         alert('HTML: '+$('.world').html());
         alert('Value: '+$('.test').val());
        });
        });

    </script>
    <!-- <script src="/js/jquery-latest.js"></script> -->
</html>


第一次alert弹出<p></p>里的文本内容不包含html标签，第二次alert会弹出带有<p></p>包裹的文字，第三次弹出为表单控件的value值，val为value的缩写

设定内容：

<html>
    <head>
        <title>JQuery DOM Manipulation</title>
    </head>
    <body>
        <p id="test1">this is the first test</p>
        <p id="test2">this is the other test</p>
        <p>Input field: <input type="text" id="test3" value="handsome Haris"></p>
        <button id="btn1">设置文本</button>
        <button id="btn2">设置 HTML</button>
        <button id="btn3">设置值</button>
    </body>
    <script src="../../js/jquery-3.2.1.min.js"></script>

    <script type="text/javascript">
        $("document").ready(function() {
            $("#btn1").click(function() {
                $("#test1").text("Hello world!");
            });
            $("#btn2").click(function() {
                $("#test2").html("<b>Hello world!</b>");
            });
            $("#btn3").click(function() {
                $("#test3").val("are you kidding me?");
            });
        });
    </script>
    <!-- <script src="/js/jquery-latest.js"></script> -->
</html>

attr() and removeAttr()

attr() 方法用于获取属性值： $(selector).attr(attribute)
removeAttr() 方法从被选元素中移除属性：$(selector).removeAttr(attribute)

<html>
    <head>
        <title>JQuery DOM Manipulation</title>
    </head>
    <body>
        <p><a href="http://baidu.com" id="baidu">baidu.com</a></p>
        <span style="font-size:120%;color:red">这是一个段落。</span>
        <button class="btn1">显示 href 值</button>
        <button class="btn2">删除span元素的 style 属性</button>
    </body>
    <script src="../../js/jquery-3.2.1.min.js"></script>

    <script type="text/javascript">
        $("document").ready(function() {
            $(".btn1").click(function() {
                alert($("a").attr("href"));
            });
            $(".btn2").click(function() {
                $("span").removeAttr("style");
            });
        });
    </script>
    <!-- <script src="/js/jquery-latest.js"></script> -->
</html>


设置格式：$(selector).attr('attribute name','value')

css() 、addClass()、removeClass() and toggleClass()

  • css() - 设置或返回样式属性
  • addClass() - 向被选元素添加一个或多个类
  • removeClass() - 从被选元素删除一个或多个类
  • toggleClass() - 对被选元素进行添加/删除类的切换操作

==toggle==就像一个开关，有类就删除，无类则添加

css()方法：
css(“propertyname”) 返回指定的 CSS 属性的值
css(“propertyname”,”value”) 设置指定的 CSS 属性
css({“propertyname”:”value”,”propertyname”:”value”,…})
设置多个 CSS 属性

css()方法就不举例了

<html>
    <head>
        <title>JQuery DOM Manipulation</title>
        <style>
            .move
                    {
                    background-color: yellow;
                    }
            .blue
                    {
                    color:blue;
                    }
        </style>
    </head>
    <body>
        <h1>标题 1</h1>
        <h2>标题 2</h2>
        <p>这是一个段落</p>
        <p>这是另一个段落</p>
        <div class="move">去除这个黄色类</div>
        <br>
        <button class="btn1">向元素添加类</button>
        <button class="btn2">移出元素的类</button>
        <button class="btn3">toggle测试开关</button>
    </body>
    <script src="../../js/jquery-3.2.1.min.js"></script>

    <script type="text/javascript">
        $("document").ready(function() {
            $(".btn1").click(function() {
                $("h1,h2,p").addClass("blue");
            });
            $(".btn2").click(function() {
                $("div").removeClass("move");
            });
            $(".btn3").click(function() {
                $("div").toggleClass("move");
            });
        });
    </script>
    <!-- <script src="/js/jquery-latest.js"></script> -->
</html>

append() and prepend() | appendTo() and prependTo() | before() and after() | insertBefore() and insertAfter()

append()和appendTo()、prepend()和prependTo()这两对实际上是在玩文字游戏
append() - 在被选元素的结尾插入内容
例：$('p').append('text')
appendTo() - 将内容插入到元素结尾
例：$('text').append('p')
prepend() - 在被选元素的开头插入内容
例：$('p').prepend('text')
prependTo() - 将内容插入到元素开头
例：$('text').prepend('p')
after() - 在被选元素之后插入内容
before() - 在被选元素之前插入内容
insertBefore() - 把匹配的元素插入到另一个指定的元素集合的前面
insertAfter() - 把匹配的元素插入到另一个指定的元素集合的后面

$(content).insertBefore(selector)
$(content).insertAfter(selector)

全部例子：

<html>
    <head>
        <title>JQuery DOM Manipulation</title>
    </head>
    <body>
        <p id="i1">This is a paragraph.</p>
        <p id="i2">This is another paragraph.</p>
        <ol>
            <li>List item 1</li>
            <li>List item 2</li>
            <li>List item 3</li>
        </ol>
        <p><span class="s1">前</span><span class="s2">后</span></p>
        <div>
            <div id="1">
                1
            </div>
            <div id="2">
                2
            </div>
        </div>
        <br/><br/>
        <button id="btn1">追加文本</button>
        <button id="btn2">追加列表项</button>
        <br/><br/>
        <button id="btn3">before</button>
        <button id="btn4">after</button>
        <br/><br/>
        <button id="btn5">insertBefore</button>
        <button id="btn6">insrtAfter</button>
    </body>
    <script src="../../js/jquery-3.2.1.min.js"></script>

    <script type="text/javascript">
        $("document").ready(function() {
                    $("#btn1").click(function() {
                        $("#i1,#i2").append("<b>Appended text</b>.");
                        $("#i1,#i2").prepend("<b>Prepended text</b>.");
                    });

                    $("#btn2").click(function() {
                        $("ol").append("<li>Appended item</li>");
                        $("ol").prepend("<li>Prepended item</li>");
                    });
                    $("#btn3").click(function() {
                        $(".s1").before("<span>在前之前插入的文本</span></br>");
                    });
                    $("#btn4").click(function() {
                        $(".s2").after("</br><span>在后之后插入的文本</span>");
                    });
                    $("#btn5").click(function() {
                            $("<div>3</div>").insertBefore("#1");
                            });
                    $("#btn6").click(function() {
                            $("<div>4</div>").insertAfter("#2");
                            });
                    });
    </script>
    <!-- <script src="/js/jquery-latest.js"></script> -->
</html>

remove()、empty() and detach()

通过 jQuery，可以很容易地删除已有的 HTML 元素

  • remove() - 删除被选元素（及其子元素）
  • empty() - 从被选元素中删除子元素
  • detach() - 移除被选元素，包括所有文本和子节点

detach() 会保留所有绑定的事件、附加的数据，这一点与 remove() 不同, $(selector).detach()

remove()方法不会把匹配的元素从 jQuery 对象中删除，因而可以在将来再使用这些匹配的元素,但除了这个元素本身得以保留之外，remove() 不会保留元素的 jQuery 数据,其他的比如绑定的事件、附加的数据等都会被移除,$(selector).remove()

很简单不举例了

unwrap()、wrapInner()、wrap() and wrapAll()

  • wrap() - 把匹配的元素用指定的内容或元素包裹起来
  • unwrap() - 移除并替换指定元素的父元素
  • wrapAll() - 把所有匹配的元素用指定的内容或元素包裹起来
  • wrapInner() - 将每一个匹配的元素的子内容用指定的内容或元素包裹起来
$(".btn1").click(function(){
   $("p").wrap("<div></div>");
});

<p>标签外层再套一层<div>

$("button").click(function(){
  $("p").unwrap();
});

与wrap相反，去掉<p>标签外的标签

$(".btn1").click(function(){
   $("p").wrapAll("<div></div>");
});

与wrap的不同点就是他可以将许多<p>都一同套上<div>

$(".btn1").click(function(){
   $("p").wrapInner("<b></b>");
});

将<p>标签里的内容套上<b>

replaceWith() and replaceAll()

replaceWith() 方法用指定的 HTML 内容或元素替换被选元素

$(selector).replaceWith(content)

replaceAll() 方法用指定的 HTML 内容或元素替换被选元素

$(content).replaceAll(selector)

显而易见，这两个方法的作用是相同的，区别在于内容和选择器的位置

clone()

clone() 方法生成被选元素的副本，包含子节点、文本和属性

$(selector).clone(includeEvents)



<html>
    <head>
        <title>JQuery DOM Manipulation</title>
    </head>
    <body>
        <p class="p1">这个段落是append直接操作</p>
        <p class="p2">这个段落是append操作克隆过</p>
        <button id="bt1">直接append内容在body后</button>
        <br/>
        <br/>
        <button id="bt2">append克隆后的内容在body后</button>
    </body>
    <script src="../../js/jquery-3.2.1.min.js"></script>

    <script type="text/javascript">
        $("document").ready(function() {
            $("#bt1").click(function() {
                $("body").append($(".p1"));
            });
            $("#bt2").click(function() {
                $("body").append($(".p2").clone());
            });
        });
    </script>
    <!-- <script src="/js/jquery-latest.js"></script> -->
</html>


==注意==：
append操作相当于是“剪切—粘贴”的操作，如图，它是将原来的<p>剪切下来后加在了body后
经过克隆过后的<p>它每触发一次按钮点击，都会进行一次复制与粘贴的过程，并且复制过程中基数会是上一次的个数
